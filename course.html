What is the MEAN stack?


MEAN stack is a JavaScript-based framework for developing web applications.
MEAN is named after MongoDB, Express, Angular, and Node, the four key technologies that make up the layers of the stack.

  1. MongoDB — document database
  2. Express(.js) — Node.js web framework
  3. Angular(.js) — a client-side JavaScript framework
  4. Node(.js) — the premier JavaScript web server

What is the MERN stack?


MERN stands for MongoDB, Express, React, Node, after the four key technologies that make up the stack.

  1. MongoDB — document database
  2. Express(.js) — Node.js web framework
  3. React(.js) — a client-side JavaScript framework
  4. Node(.js) — the premier JavaScript web server

What is MEVN?

MEVN is again a web stack just like MERN and MEAN though it is relatively new as the front-end technology used in it which is Vue.JS.

    Advantages of VueJS:
  1. It is platform-independent.
  2. Vue.JS is faster and easy to learn.
    Disadvantages of VueJS:
  1. Vue.JS is a new framework hence it doesn't have the support of a large community.
  2. Lack of plugins as Vue is still not as big as Angular or React.
